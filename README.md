DisableFrostWalker
==================
氷渡りエンチャントを無効化するBukkitプラグイン

Commands
--------
* /disablefrostwalker(/dfw) reload - config.ymlを再読み込み

Permissions
-----------
* disablefrostwalker.reload - /dfw reloadの実行

Configuration
-------------
\# 氷渡りを有効化するワールド名を指定(正規表現)  
\# default: ["world"]  
enableWorlds: ["world"]
